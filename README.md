# dashboard-powerbi-ventas-inventario
# 📊 Dashboard Interactivo – Análisis de Ventas e Inventario

## 🎯 Objetivo del proyecto
Desarrollar un dashboard interactivo que permita analizar el desempeño de ventas e inventario, facilitando la toma de decisiones basada en datos.

---

## 📈 Descripción

Este proyecto fue desarrollado en Power BI y permite:

- Analizar ventas en el tiempo
- Evaluar niveles de inventario
- Identificar productos con riesgo de quiebre de stock
- Relacionar unidades vendidas con stock disponible

---

## 🧠 Principales insights

- Existe una relación directa entre ventas y necesidad de reposición de stock  
- Se identifican productos con bajo stock en relación a su demanda  
- La gestión de inventario puede optimizarse mediante el análisis de datos  

---

## 📊 Visualizaciones utilizadas

- Gráficos de líneas (tendencias)
- Gráficos de barras (comparaciones)
- Indicadores KPI
- Tablas con formato condicional

---

## 🛠️ Herramientas

- Power BI
- Modelado de datos
- DAX
- Visualización de datos

---

## 🚀 Valor del proyecto

Este dashboard permite transformar datos operacionales en información útil para la toma de decisiones, especialmente en la gestión de inventario y ventas.
